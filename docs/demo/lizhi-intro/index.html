Vault · 03加密库

本地优先

创建、解锁与落盘加密

  • 新建知识库：设定主密码，生成 vault 与密钥材料
  • 解锁会话：密码派生密钥；密钥只在 Rust 侧持有，不进前端持久化
  • 文档与资产：正文加密存储，元数据进本地数据库
  • 算法基线：AES-256-GCM + Argon2id（可在设置/白皮书口径对齐）
  • 锁定：离开时可再次锁定，锁屏遮罩而非直接丢窗
怎么用 首次 Welcome 创建库 → 之后每次从解锁页进入 → 侧栏可随时锁定
Workspace · 04写作与组织

核心日常

工作区：文档、文件夹、搜索

  • Markdown 编辑：CodeMirror 源码编辑，自动保存
  • 文件夹树：新建 / 重命名 / 移动文档；命令面板快捷操作
  • 搜索：库内检索标题与正文，快速跳转
  • 标签：文档打标，侧栏与库状态同步
  • 视图切换：编辑视图 ↔ 局部图谱视图（同在工作区）
  • 附件资源：资产目录存图等材料，写入加密资产区
入口 侧栏「工作区」→ 点选文档开始写；Ctrl/Cmd+K 开命令面板
Network · 05双链与图谱

把笔记织成网

双链、反链、局部图谱

写作时
Wiki 双链
  • 输入 [[ 触发搜索建议（含拼音）
  • Cmd/Ctrl+Click 跳转目标文档
  • 悬浮预览链接内容
  • 写入时增量更新 links 索引
浏览时
反链 · 图谱
  • 查看谁链到了当前文档
  • 未链接提及可转成正式双链
  • 局部图谱约 2-depth 力导向布局
  • 工作区内一键切换图谱视图
Insights · 06洞察看板

默认首页

热力图与库概览

  • 写作热力图：按日编辑活动，看长期节奏
  • 库统计：文档量、链接概况等仪表信息
  • 最近动态：从活动中快速回到正在推进的笔记
  • 作为启动页：解锁后默认进入洞察，再下钻工作区
入口 解锁成功 → /insights；也可随时从侧栏回洞察
Security · 07安全与导出

防窥与外发

App Lock、水印、导出清理

01 / LOCK

应用锁定

离开座位一键锁定。设置页开关；平台暂不支持的能力会明确提示。

02 / WATERMARK

界面 / 导出水印

Canvas 覆盖水印可配；导出 PDF/长图时可带水印，默认关闭、推荐按场景开启。

03 / EXPORT

MD / PDF 导出

导出时可清洗元数据；敏感外发前可先预览与确认。

AI · 08灵狸助手

应用内对话

三种助手模式

  • 闲聊：普通问答，不强制读库
  • RAG：基于知识库检索回答，引用你的笔记
  • 笔记助手：围绕当前文档改写、总结、扩展
  • 配置：在设置中接入模型与密钥；工具调用过程可在聊天区展示
  • 边界：仅在 vault 解锁后可用，避免明文会话越权
入口 侧栏「AI」→ /ai；先到设置完成模型配置
Agent · 09工作台与 MCP

给 Agent 用的能力面

Agent 工作台 + lizhi-mcp

Agent 工作台
/cc-workbench
  • 基于 Claude Agent SDK 的独立工作台
  • 工作目录可选 vault（默认）或 project
  • vault 模式侧重知识库工具，而非裸盘乱写
  • 与应用内 AI 助手并列，互不合并
MCP
lizhi-mcp
  • 列出 / 读取 / 搜索 / 保存文档
  • 文件夹树、标签、反链、图谱快照
  • 设置页管理 MCP 服务启用与配置
  • 改包后需 build + sync 进 Tauri 资源
Modules · 10专项模块

业务向扩展页

日记 · 凭证 · 需求 · 发布

日记

/journal — 日记式记录，与普通文档流分离的日常入口。

凭证

/credentials — 敏感条目集中存放，减少散落在笔记里的明文口令。

需求

/requirements — 需求条目管理，适配产品/项目笔记场景。

发布记录

/launches — 发布卡片、时间线与抽屉详情，追踪上线节奏。

备份扩展

设置内备份：整库 replace / 设置 merge / 文档合并等恢复模式。

UI 状态

文件夹展开、标签、对话等写入 vault-ui-state，Tauri 下双写。
