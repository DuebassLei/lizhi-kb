## 1.5 视觉哲学（巢域静默）

**巢域静默**是一种将私密性视为空间而非装饰的美学运动。知识不是陈列在橱窗里的商品，而是栖息在巢穴深处的生命体——它需要黑暗、边界与警觉的守卫。这一哲学拒绝霓虹黑客风与低幼卡通，也拒绝军事化的恐吓营销；它追求的是博物馆级克制：像守夜的猫一样安静，像密码学家一样可验证。

空间与形式在此扮演首要角色。构图以巢形几何为隐性骨架——圆弧庇护、节点眼瞳、三向联结线——但从不直白图解产品功能。负空间占据画面六成以上，让「未言说」成为信息本身。元素沿 8px 网格精密对齐。

色彩与材质遵循墨巢底色 `#1A1D23`，联结蓝 `#5b9fd4` 如远处微光，暖爪色 `#d4a574` 作为极少量的生命脉搏——不超过画面 8% 面积。材质感偏向磨砂玻璃与雾面金属；蓝域暗示联结与图谱，暖爪暗示品牌温度，墨巢暗示加密与本地。

尺度与节奏遵循呼吸式递进。主标题以克制的大字号锚定画面，辅文以细小无衬线标签漂浮于系统图旁。构图视觉重心略偏左下，模拟巢穴入口的邀请感；联结线图谱作为低透明度背景纹理。整体追求不可再减的纯粹，令观者感到思想的绝对主权被认真对待与守护。

- 网格：8px 基准

- 按钮圆角：8px
- 最大内容宽度：1120px

### 2.4 组件状态

| 组件 | default | hover | focus | disabled |
|------|---------|-------|-------|----------|
| 主按钮 | bg paw, text ink | brightness 1.08 | ring 2px paw | opacity 0.5 |
| 次按钮 | border, transparent | bg surface | ring link | opacity 0.5 |
| 徽章 | mono 11px, secure/link | — | — | — |

### 2.5 无障碍

- 正文对比度 ≥ 4.5:1（WCAG AA）
- 所有交互元素可键盘聚焦，可见 focus ring
- 动效尊重 `prefers-reduced-motion`
- 图标按钮附带 `aria-label`

### 4.3 功能三列

#### 列 1 · 加密巢穴

**标题**：落盘即加密，解锁才明文

**正文**  
所有笔记以 AES-256-GCM 加密存储。主密码经 Argon2id 派生密钥，抗 GPU 暴力破解。锁定后内存中的密钥与明文主动清零——不是关窗，是模糊遮罩，思考不断线。

**要点**
- Argon2id · 64 MB · AES-256-GCM
- 生物识别快捷解锁
- 恢复密钥 PDF 导出

#### 列 2 · 知识成网

**标题**：双链呼吸，图谱可见

**正文**  
输入 `[[` 自动补全，反向链接实时更新。局部图谱以力导向布局呈现 2 层深度联结。知识不再困在文件夹里——它开始自己说话。

**要点**
- Wiki 双链 + 拼音搜索
- 反向链接 / 未链接提及
- 局部图谱 ≥ 55fps

#### 列 3 · 防窥守护

**标题**：界面水印，导出溯源

**正文**  
Canvas 覆盖层防旁人瞟屏；PDF 导出可强制嵌入水印。元数据自动清洗——无 Author、无 Creator、无 EXIF。每次导出显示「已清洗元数据 ✓」。

**要点**
- 可配置界面水印
- 导出水印 + 元数据剥离
- 防暴破阶梯延迟

### 4.9 FAQ

**Q: 狸知是 Obsidian 插件吗？**  
A: 不是。狸知是独立桌面应用，从存储层实现加密，Tauri + Rust 构建。

**Q: 真的零网络吗？**  
A: 默认 Rust 层拦截所有出站连接。设置页可查看实时拦截计数。更新需手动下载。

**Q: 忘记主密码怎么办？**  
A: 凭恢复密钥可重置。无恢复密钥则无法解密——这是加密的设计使然，不是缺陷。

**Q: 支持移动端吗？**  
A: v1.x 仅桌面端（Win / macOS / Linux）。移动端在路线图中，暂无日期。

**Q: 能导入 Obsidian 库吗？**  
A: 支持导入 Markdown 文件夹，保留 `[[双链]]` 语法。

**Q: 真的完全免费吗？**  
A: 是。狸知不设付费版本，所有功能免费使用，无订阅、无内购。

## 5. 产品介绍（长文版）

### 5.1 一句话

**狸知知识库**是本地优先、端到端加密的个人知识库——把 Obsidian 的知识网络，装进真正加密落盘的猫窝里。

### 5.2 产品定义

狸知（Lizhi Knowledge）面向需要**长期知识沉淀**且对**数据主权**有强需求的个人用户。它不是云端笔记，不是协作白板，而是一个完全运行在本地、默认零网络、内容落盘即加密的深度思考环境。

品类：**个人加密知识库**（Encrypted Personal Knowledge Base）

### 5.3 核心价值

1. **绝对数据主权** — 思想只属于你。无云依赖、无厂商锁定、无隐式同步。
2. **可验证本地加密** — AES-256-GCM 落盘、Argon2id 派生、网络拦截可计数。
3. **知识网络化** — 双链、反向链接、局部图谱，让笔记从「文件」变成「网络」。
4. **防窥与溯源** — 界面水印、导出水印、元数据清洗，面向真实物理世界威胁。

### 5.4 产品版本

| 版本 | 代号 | 状态 | 核心能力 |
|------|------|------|----------|
| v1.0 | Vault | 已交付 | 加密库、编辑器、目录树、热力图、MD/PDF 导出 |
| v1.5 | Network | 当前 | 双链、图谱、App Lock、界面/导出水印 |
| v2.0 | Shadow | 规划 | 诱饵库、盲水印、防截屏、开源加密核心 |

### 5.5 技术栈摘要

- **桌面壳**：Tauri 2
- **后端**：Rust（加密、存储、网络隔离）
- **前端**：Vue 3 + Pinia + Tailwind CSS 4
- **编辑器**：CodeMirror 6 / Markdown 双模式
- **图谱**：SVG + composable 力导向布局

### 5.6 品牌叙事

吉祥物**守夜**——几何黑猫，象征安静、警觉、私密。主 Slogan「你的加密知识库，猫一样安静」传递三重含义：不打扰思考、不向外窥探、夜间守夜般的警觉。
