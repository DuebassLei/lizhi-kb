# 狸知知识库 — 品牌方案（小猫 × 加密知识库）

**日期**：2026-07-06  
**状态**：已定稿  
**产品全称**：狸知知识库 · Lizhi Knowledge  
**品类**：本地加密个人知识库 / Encrypted Personal Knowledge Workbench  
**调性约束**：专业可信（研究员、顾问）+ 温暖私密（猫的陪伴感），**不要**低幼卡通、不要黑客骷髅风  

---

## 1. 命名策略

好名字需同时满足：

| 维度 | 要求 |
|------|------|
| 语义 | 知识、联结、本地、加密 至少命中其二 |
| 猫元素 | 可抽象（须、爪、尾、夜）不必直白「喵喵」 |
| 发音 | 中文 2–3 字好记；英文便于域名与 Tauri 包名 |
| 差异化 | 避开 CatNote、Kitty 等泛名与已有笔记产品 |
| 扩展 | 可作动词（「记进狸知」）、图标（猫头/猫尾曲线） |

## 2. 候选方案（五选一）

### 🏆 方案 A：**狸知 Lizhi**（已选定 · v2 强化知识库识别）

| 要素 | 内容 |
|------|------|
| **品牌名（短）** | **狸知** — Logo、日常口头称呼 |
| **产品全称（中文）** | **狸知知识库** — 官网、商店、标题栏、首次介绍 **必用** |
| **产品全称（英文）** | **Lizhi Knowledge** — 副标 *Encrypted Knowledge Base* |
| **品类一行** | **个人加密知识库** — 始终紧贴品牌下方（≤12 字） |
| **包名 / 目录** | `lizhi-kb` / `com.lizhi.knowledge` |
| **命名故事** | 「狸」猫之原型；「知」知识；全称补「**库**」— 明确是 **Knowledge Base** 而非泛笔记/日记。谐音「理智」保留。 |
| **Slogan（主）** | **你的加密知识库，猫一样安静。** |
| **Slogan（联结）** | **私密如猫，知识成网。** / *Private as a cat. Knowledge as a web.* |
| **Slogan（安全）** | **库在本地，爪护全文。** |

**双层命名规则**：

```
对外首次触达  →  「狸知知识库」+ 个人加密知识库
老用户 / 图标  →  「狸知」
英文          →  Lizhi Knowledge（不写 Lizhi Note）
```

**优点**：保留原方案气质；「库」字显性，与 Obsidian/Notion 品类对齐。  
**风险**：与「荔枝」同音 — 英文 **LiZhi Knowledge** 区分；视觉用猫耳非水果。

## 4. 视觉识别（VI）— 猫 × Vault Noir

### 4.1 调性关键词

**安静、警觉、柔软、精确** — 像深夜写作时卧在键盘旁的猫，不是 Pet App。

### 4.2 色彩（在原型 Vault Noir 上微调）

| Token | 色值 | 用途 |
|-------|------|------|
| 墨巢 `--ink` | `#090b0e` | 背景 |
| 联结蓝 `--link` | `#5b9fd4` | 双链、图谱 |
| 守护绿 `--secure` | `#3ecf9a` | 加密、解锁 |
| **暖爪 `--paw`** | `#d4a574` | **猫品牌点缀**（logo、空状态、成就徽章） |
| 须尖 `--whisker` | `#8a9bb0` @ 40% | 分割线、水印次要色 |

### 4.3 Logo 概念：**「巢中眼」**

```
        ╱╲
       ╱  ╲      ← 简化猫耳（两三角）
      │ ◉  │     ← 单眼 / 节点（知识焦点）
       ╲__╱
      ╱    ╲     ← 底部弧线 = 猫窝 / vault 盾形
     ○──○──○     ← 三节点 = 双链网络（可选）
```

- 单线稿，1.5px stroke，可单色反白
- 不用整只卡通猫，避免幼稚
- App 图标：深色底 + 暖爪色猫耳 + 蓝绿节点

### 4.4 吉祥物「守夜」（可选）

| 属性 | 设定 |
|------|------|
| 名字 | **守夜**（Night） |
| 形象 | 几何折纸风黑猫剪影，仅眼睛与耳尖用 `--paw` 点缀 |
| 出现场景 | 空文档、FTUE 最后一屏、连续写作 streak 成就、错误页（「守夜发现 vault 已锁定」） |
| 禁止 | 卖萌表情包、对话气泡刷屏 |

### 4.5 字体

| 用途 | 字体 |
|------|------|
| 中文标题 | **思源宋体 / 霞鹜文楷**（知识感） |
| 英文 / UI | **Sora**（与原型一致） |
| 代码 | IBM Plex Mono |
