FAQ

常见问题

狸知是 Obsidian 插件吗？
不是。狸知是独立桌面应用，从存储层实现加密，基于 Tauri + Rust 构建。
真的零网络吗？
默认 Rust 层拦截所有出站连接。设置页可查看实时拦截计数。更新需手动下载。
忘记主密码怎么办？
凭恢复密钥可重置。无恢复密钥则无法解密——这是加密的设计使然，不是缺陷。
能导入 Obsidian 库吗？
支持导入 Markdown 文件夹，保留 [[双链]] 语法。
真的完全免费吗？
是。狸知不设付费版本，所有功能免费使用，无订阅、无内购、无试用到期。
